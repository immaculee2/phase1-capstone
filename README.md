 Book Explorer

1 Responsive & Interactive Web App**

Book Explorer is a web application that lets users **browse, search, and favorite books** fetched from the Gutendex API. Favorites are stored in **localStorage**, and the app is fully **responsive** with multi-page navigation (Home, Favorites, About).


2 Features

- Browse live book data  
- Search books by title  
- Add/remove favorites with persistence  
- Responsive design for mobile, tablet, and desktop  
- Multi-page navigation

 Technologies

HTML5, Tailwind CSS, JavaScript (ES6), Fetch API, localStorage, Git & GitHub
